Project Description:

The project involved orchestrating infrastructure and automating deployment on AWS using Terraform and Jenkins, and enhancing communication through Amazon Simple Email Service (SES).

The project involved creating separate production and deployment workspaces with variable definitions, employing Terraform modules for network resource management, and deploying two AWS environments. Additionally, I configured a Jenkins Docker image with Terraform to automate deployments and implemented a flexible Terraform code application pipeline. We also integrated Amazon SES for email verification and utilized a Python Lambda function to send email notifications triggered by changes in the S3 state file.
